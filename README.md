## 法務RPG

### 起動方法

1. `C:\Users\akita\OneDrive\ドキュメント\仕事\legal-rpg` を開く
2. `index.html` をブラウザで開く
3. 「コースを選択」から受講するコースを選ぶ（例：総合コース / 個人情報保護法コース）

### 研修結果の自動回収（Googleスプレッドシート）

セッション終了・全クリア時に、結果画面に「📝 結果を送信」ボタンを表示してスプレッドシートへ記録できます（事前設定が必要）。

手順（概要）：
1. Googleスプレッドシートを作成
2. Apps Script を開き、`scripts/gas_result_receiver.gs` の内容を貼り付け
3. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」
   - 実行するユーザー：自分
   - アクセスできるユーザー：社内（または必要に応じて「全員」）
4. WebアプリURLをコピーし、`app.js` の `RESULT_POST_URL` に貼り付け

### 問題の編集方法

1. `admin.html` をブラウザで開く
2. `questions.json` をドラッグ&ドロップ（またはクリックして選択）
3. 左のリストから編集したい問題を選ぶ
4. 問題文・選択肢・正答・解説を直接入力
5. 「💾 JSONをダウンロード」ボタンを押す
6. ダウンロードされた `questions.json` を元のファイルに上書き保存

**編集できる項目:** 問題文 / 選択肢(4択) / 正答 / 解説 / テーマ / コース（任意） / タグ / ステージ（1〜6） / Fatal設定

**その他の操作:**
- 検索・フィルター: コース・テーマ・ステージで問題を絞り込み
- 問題追加: 右上「＋ 問題追加」ボタン
- 問題削除: 編集パネル右上「削除」ボタン

---

### 実装済み

- プレイヤー名入力
- ステータス表示（HP/EXP/LV/役職）
- 3ステージ（初級/中級/上級）各3問
- 正答: EXP +10 / 誤答: HP -20
- EXPが一定値に達するとレベルアップ・役職昇格
- HP0以下でゲームオーバー
- 装備/アイテム
  - 六法（ヒント+1）
  - 判例DB（選択肢の一部除外）
  - 生成AIアシスト（不正解時のHP減少を軽減）
- ヒント機能（回数制限あり）
- `localStorage` による途中保存/再開
